Acceso administrador

Ruta del panel:

/admin.html

El correo y contraseña se administran desde Supabase Auth. No compartir la clave por canales públicos.

Uso diario

  • Crear o editar productos.
  • Subir imagen o pegar URL directa.
  • Revisar pedidos nuevos.
  • Confirmar pedido para descontar stock.
  • Actualizar stock manualmente.

Seguridad

  • Cambiar contraseña al entregar el sitio.
  • Usar correo administrador único.
  • Cerrar sesión al terminar.
  • No publicar claves secretas de Supabase.

Imágenes

Para carga directa desde el panel, ejecutar primero supabase_storage_setup.sql. Como respaldo, se puede pegar una URL pública o una ruta del sitio.
