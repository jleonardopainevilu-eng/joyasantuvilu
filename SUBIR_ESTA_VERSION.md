# SUBIR ESTA VERSIÓN

Esta versión está lista para reemplazar la actual en GitHub.

Orden recomendado:

1. Subir todos los archivos del ZIP a la raíz del repositorio.
2. Esperar deploy en Vercel.
3. Abrir `https://www.joyasantuvilu.cl/`.
4. Debe verse la tienda con portada, beneficios, catálogo en preparación y videos.
5. Abrir `https://www.joyasantuvilu.cl/admin.html`.
6. Entrar con el usuario de Supabase.
7. Crear 3 productos reales desde el panel.
8. Volver a la tienda pública y recargar.

Si quieres probar productos antes de cargar los reales, ejecuta `supabase_seed_productos_prueba.sql` en Supabase SQL Editor con rol `postgres`.
