# Producto: Tienda Online Boutique

## Posicionamiento

Una tienda online boutique es mas que una pagina bonita y menos pesada que un ecommerce corporativo grande.
Esta pensada para negocios con productos fisicos, stock acotado y venta directa.

## Incluye en version base

| Modulo | Incluido |
|---|---|
| Home comercial | Si |
| Catalogo responsive | Si |
| Buscador y filtros | Si |
| Ficha de producto | Si |
| Carrito | Si |
| Pedido manual | Si |
| Panel admin | Si |
| Login seguro | Si |
| Stock en base de datos | Si |
| Pedidos guardados | Si |
| Descuento de stock al confirmar | Si |
| Historial inventario | Si |
| Pago online | No, modulo futuro |
| Envios avanzados | No, modulo futuro |
| Reportes avanzados | No, modulo futuro |

## Entrega estandar al cliente

- Sitio responsivo.
- Panel `/admin`.
- Carga inicial de productos.
- Configuracion de correo y WhatsApp.
- Capacitacion corta para administrar productos, stock y pedidos.
- Documentacion de acceso.

## Extras cobrables

- Pago online.
- Integracion Webpay / Flow / Mercado Pago.
- Carga masiva de productos.
- Correos automaticos.
- Reportes de ventas.
- Roles adicionales.
- Diseno de identidad visual.
- Produccion de fotos o videos.
- Mantencion mensual.

## Diferencia frente a una web de bajo costo

Una web simple normalmente trae contacto, secciones y WhatsApp.
Esta base trae logica de negocio: pedidos, stock, login, inventario e historial.
